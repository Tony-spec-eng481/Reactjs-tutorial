Functions & JSX

📖 JavaScript in JSX

You can put any valid JavaScript expression inside curly braces in JSX. For example, 2 + 2, user.firstName, or formatName(user) are all valid expressions.

function formatName(user) {
  return user.firstName + ' ' + user.lastName;
}

const user = {
  firstName: 'Harper',
  lastName: 'Perez'
};

const element = (
  <h1>
    Hello, {formatName(user)}!
  </h1>
);
user ➔ formatName(user)
