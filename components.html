React Components

🎯 What are Components?

Components are the building blocks of React applications. They let you split the UI into independent, reusable pieces.

Types of Components

1. Functional Components

Functional components are JavaScript functions that return JSX.

function Greeting() {
  return <h1>Hello from Functional Component!</h1>;
}

2. Class Components (Legacy)

Class components use ES6 classes and have additional features like lifecycle methods.

class Greeting extends React.Component {
  render() {
    return <h1>Hello from Class Component!</h1>;
  }
}

Component Composition

Components can refer to other components in their output.

function Welcome() {
  return <h1>Welcome!</h1>;
}

function App() {
  return (
    <div>
      <Welcome />
      <Welcome />
      <Welcome />
    </div>
  );
}
📌 Important Rules:
  • Component names must start with a capital letter
  • Components can only return one parent element
  • Components should be pure functions (no side effects)

Component Hierarchy Example

// Parent Component
function Header() {
  return <h1>My Website</h1>;
}

// Child Component
function Navbar() {
  return (
    <nav>
      <a href="/">Home</a>
      <a href="/about">About</a>
    </nav>
  );
}

// App Component (Parent of all)
function App() {
  return (
    <div>
      <Header />
      <Navbar />
    </div>
  );
}
