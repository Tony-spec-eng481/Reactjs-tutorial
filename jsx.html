JSX in Depth

💎 What is JSX?

JSX is a syntax extension to JavaScript. It looks like HTML but has the full power of JavaScript.

JSX Syntax

const element = (
  <h1 className="greeting">
    Hello, world!
  </h1>
);

Compiled JavaScript

const element = React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);
