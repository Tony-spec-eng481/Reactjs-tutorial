Event Handling

⚡ Handling Events in React

Handling events with React elements is very similar to handling events on DOM elements. However, there are some syntactic differences.

Key Differences

  • React events are named using camelCase, rather than lowercase (e.g., onClick instead of onclick).
  • With JSX you pass a function as the event handler, rather than a string.
function ActionLink() {
  function handleClick(e) {
    e.preventDefault();
    console.log('The link was clicked.');
  }

  return (
    <a href="#" onClick={handleClick}>
      Click me
    </a>
  );
}
